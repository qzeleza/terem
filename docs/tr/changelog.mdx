# Sürüm Geçmişi

## [v1.2.0] - 2024-01-15

### 🎉 Yeni Özellikler
- ASUS RT-AX89X yönlendirici desteği eklendi
- Otomatik güncelleme işlevi uygulandı
- Sıcaklık izleme modülü eklendi
- Grafikler için yeni TUI bileşeni

### 🐛 Hata Düzeltmeleri
- Standart olmayan portlara sahip yönlendiriciler için SSH bağlantı hatası düzeltildi
- Sistem bilgi modülündeki bellek sızıntısı giderildi
- VLAN arayüzleri için MAC adresi ayrıştırma düzeltildi

### 📦 Bağımlılıklar
- termos sürüm 1.2.0'a güncellendi
- zlogger sürüm 1.0.2'ye güncellendi
