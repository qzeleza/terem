# Настройка

После установки Терем необходимо выполнить базовую настройку приложения.

## Создание конфигурации

```bash
# Генерация базовой конфигурации
terem init

# Редактирование конфигурационного файла
vi /etc/terem/config.yaml
```

## Основные параметры

```yaml
# /etc/terem/config.yaml
app:
  title: "Терем"
  devMode: false
  debug: false

log:
  level: "info"
  file: "/tmp/terem.log"

network:
  router:
    address: "192.168.1.1"
    arch: "arm"
    platform: "openwrt"
    sshPort: "22"
```

## Проверка параметров

```bash
# Проверка конфигурации
terem config validate

# Просмотр текущих настроек
terem config show
```

## Настройка логирования

```bash
# Изменение уровня логирования
terem log level debug

# Просмотр логов
terem log tail
```

## Настройка доступа к роутеру

```bash
# Проверка доступности роутера
terem router ping

# Проверка SSH подключения
terem router ssh-test

# Проверка обязательных утилит
terem router check-utils
```

<Info>
  Убедитесь, что на роутере включен SSH-доступ и открыт порт 22 (или иной, указанный в конфигурации).
</Info>

## Импорт существующих настроек

```bash
# Импорт настроек из файла
terem config import /path/to/backup.yaml

# Экспорт текущих настроек
terem config export /path/to/terem-backup.yaml
```

## Создание пользователей (опционально)

```bash
# Создание нового пользователя
terem user add --name admin --role superuser

# Список пользователей
terem user list
```

## Рекомендации безопасности

- Смените пароль root на роутере
- Ограничьте доступ по SSH
- Включите фаервол и настройте правила
- Регулярно обновляйте прошивку и пакеты

  По завершении настройки можете перейти к разделу [Запуск](/ru/installation/launch).
