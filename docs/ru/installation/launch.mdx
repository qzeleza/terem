# Запуск Терем

## Основные режимы работы

### Интерактивный режим (TUI)
```bash
# Запуск в интерактивном режиме
terem

# Или явно
terem tui
```

### Режим команд (CLI)
```bash
# Просмотр информации о системе
terem info

# Выполнение конкретной команды
terem run [команда]
```

### Фоновый режим (Daemon)
```bash
# Запуск в фоне
terem daemon start

# Проверка статуса
terem daemon status

# Остановка
terem daemon stop
```

## Первый запуск

При первом запуске Терем покажет главное меню с опциями:
- **Системная информация** - детали о роутере
- **Приложения** - установка и настройка утилит
- **Настройки** - конфигурация Терем
- **Выход** - завершение работы

## Навигация в TUI

- **↑↓** - перемещение по меню
- **Enter** - выбор пункта
- **Esc** - назад/отмена
- **Tab** - переключение между элементами
- **F1** - справка
- **Q** - выход
