# Системные требования

Перед установкой Терем убедитесь, что ваша система соответствует минимальным требованиям.

## Минимальные требования

### Операционная система
<Tabs>
  <Tab title="OpenWrt">
    - **Версия**: OpenWrt 19.07 или выше
    - **Ядро**: Linux 4.14+
    - **Пакетный менеджер**: opkg
  </Tab>
  <Tab title="Entware">
    - **Версия**: Entware-ng или Entware 3.x
    - **Архитектура**: ARM, MIPS, x86_64
    - **Пакетный менеджер**: opkg
  </Tab>
</Tabs>

### Аппаратные требования

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| **CPU** | 400 MHz | 800 MHz+ |
| **RAM** | 64 MB | 128 MB+ |
| **Хранилище** | 10 MB | 50 MB+ |
| **Архитектура** | ARMv5+ / MIPS / x86 | ARMv7+ / MIPS 74K+ / x86_64 |

<Warning>
  На устройствах с менее чем 64 MB оперативной памяти возможны проблемы с производительностью при работе с большими объемами данных.
</Warning>

## Проверка совместимости

Выполните следующую команду для проверки совместимости вашей системы:

```bash
# Проверка версии системы
cat /etc/openwrt_release || cat /etc/entware_release

# Проверка архитектуры
uname -m

# Проверка доступной памяти
free -m

# Проверка свободного места
df -h /opt
```

## Подготовка системы

### 1. Обновление пакетов
```bash
opkg update
opkg upgrade
```

### 2. Установка базовых зависимостей
```bash
opkg install coreutils-base bash procps-ng
```

### 3. Проверка прав доступа
```bash
# Убедитесь, что у вас есть права root
whoami  # должно вывести root
```

## Сетевые требования

- **SSH доступ** - для удаленного управления
- **Интернет-соединение** - для загрузки пакетов и обновлений
- **Порты**:
  - SSH (22) - для удаленного доступа
  - Custom (по умолчанию 8888) - для веб-интерфейса (опционально)

## Рекомендации по производительности

<Steps>
  <Step title="Освободите память">
    Остановите неиспользуемые службы перед запуском Терем
    ```bash
    /etc/init.d/unnecessary_service stop
    ```
  </Step>
  <Step title="Очистите кэш">
    Регулярно очищайте системный кэш
    ```bash
    sync && echo 3 > /proc/sys/vm/drop_caches
    ```
  </Step>
  <Step title="Используйте swap">
    На устройствах с малым объемом RAM настройте swap-файл
    ```bash
    dd if=/dev/zero of=/opt/swap bs=1M count=128
    mkswap /opt/swap
    swapon /opt/swap
    ```
  </Step>
</Steps>

## Известные ограничения

- **32-битные системы** - ограничение на размер обрабатываемых файлов (до 2GB)
- **MIPS архитектура** - возможна более низкая производительность некоторых функций
- **Старые версии OpenWrt** - могут отсутствовать некоторые современные возможности
