# Работа через TUI интерфейс

TUI (Text User Interface) - основной способ взаимодействия с Терем, предоставляющий удобный графический интерфейс в терминале.

## Главное меню

При запуске `terem` вы увидите главное меню:

```
╭─ Библиотека приложений для роутера 'Терем' ─╮
│                                               │
│  > Приложения                                │
│    Настройки                                 │
│    Выход                                     │
│                                               │
╰───────────────────────────────────────────────╯
```

## Установка приложений

<Steps>
  <Step title="Выберите категорию">
    Используйте стрелки ↑↓ для навигации и Enter для выбора
  </Step>
  <Step title="Выберите приложение">
    Отметьте нужные приложения пробелом
  </Step>
  <Step title="Подтвердите установку">
    Нажмите Enter для начала установки
  </Step>
  <Step title="Дождитесь завершения">
    Прогресс-бар покажет статус установки
  </Step>
</Steps>

## Настройки

В разделе настроек доступны:

### Сетевые настройки
- Конфигурация интерфейсов
- Настройка DHCP
- Статические маршруты
- DNS серверы

### Системные настройки
- Часовой пояс
- NTP серверы
- Логирование
- Резервное копирование

### Настройки Терем
- Язык интерфейса
- Цветовая схема
- Уровень логирования
- Автообновление

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **↑/↓** | Навигация по меню |
| **←/→** | Переключение вкладок |
| **Enter** | Выбор/подтверждение |
| **Space** | Отметка элемента |
| **Esc** | Назад/отмена |
| **Tab** | Следующее поле |
| **Shift+Tab** | Предыдущее поле |
| **F1** | Справка |
| **F5** | Обновить |
| **F10/Q** | Выход |

## Работа с формами

При заполнении форм:
- Используйте Tab для перехода между полями
- Enter для отправки формы
- Esc для отмены

### Пример формы настройки
```

│                                             │
│  [Сохранить]  [Отмена]                    │
│                                             │
╰─────────────────────────────────────────────╯
```

## Индикаторы прогресса

Терем использует различные индикаторы:

- **Spinner** - для неопределенных операций
- **Progress Bar** - для операций с известным прогрессом
- **Статусные сообщения** - для быстрых уведомлений

## Цветовая индикация

- 🟢 **Зеленый** - успешное выполнение
- 🟡 **Желтый** - предупреждение
- 🔴 **Красный** - ошибка
- 🔵 **Синий** - информация
- ⚪ **Серый** - неактивный элемент

## Решение проблем

### Проблемы с отображением

```bash
# Проверьте переменную TERM
echo $TERM

# Установите правильное значение
export TERM=xterm-256color

# Проверьте локаль
locale

# Установите UTF-8
export LANG=ru_RU.UTF-8
```

### Медленная работа интерфейса

- Закройте ненужные процессы
- Увеличьте размер буфера терминала
- Используйте локальное подключение вместо SSH
