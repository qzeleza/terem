# Changelog

## [v1.2.0] - 2024-01-15

### 🎉 New Features
- Added support for ASUS RT-AX89X routers
- Implemented automatic update functionality
- Added temperature monitoring module
- New TUI component for graphs

### 🐛 Bug Fixes
- Fixed SSH connection error for routers with non-standard ports
- Fixed memory leak in system information module
- Fixed MAC address parsing for VLAN interfaces

### 📦 Dependencies
- Updated termos to version 1.2.0
- Updated zlogger to version 1.0.2
